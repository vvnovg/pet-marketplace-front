# Animal photo attributions

Each listing image in the seed (`006-seed-demo-data.yaml`) points at one of these via a root-relative `/animals/<file>` URL served by the frontend `public/animals/` directory.

Licenses: CC0 and Public domain images require no attribution; CC BY / CC BY-SA images **do** require attribution and (for BY-SA) sharing under the same license. The author and source for every file are listed below.
